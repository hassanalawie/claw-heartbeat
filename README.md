# Claw Battalion Heartbeats

This repo keeps tiny "heartbeat" files for each AI sub-agent so you can confirm they're online without wading through logs. Each file is deliberately small and can be updated or replaced whenever that agent runs.

- `heartbeats/ai-infra-scout.md`
- `heartbeats/ai-adoption-scout.md`
- `heartbeats/ai-explainability.md`
- `heartbeats/ai-creative-labs.md`

Add or remove agents as needed—every file is just a quick status snapshot.
